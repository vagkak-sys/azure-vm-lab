# Azure VM Lab

## Overview

This project demonstrates the deployment of a Windows Server virtual machine in Microsoft Azure.

## Resources Created

* Resource Group
* Virtual Network (VNET)
* Subnet
* Network Security Group (NSG)
* Public IP Address
* Windows Server 2022 Virtual Machine

## Tasks Completed

1. Created a Resource Group
2. Created a Virtual Network
3. Configured a Subnet
4. Created an NSG and allowed RDP traffic
5. Deployed a Windows Server VM
6. Connected to the VM using Remote Desktop

## Technologies Used

* Microsoft Azure
* Azure Virtual Machines
* Azure Networking
* Network Security Groups
* Remote Desktop Protocol (RDP)

## Lessons Learned

* Azure resource organization
* Virtual networking concepts
* NSG firewall rules
* Public vs Private IP addresses
* VM deployment and management
